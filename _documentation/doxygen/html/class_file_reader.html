Подробное описание

template<typename T, const uint8_t HBuffSize>
class FileReader< T, HBuffSize >

Класс для хранения параметров состояния чтения из файла

Этот класс полезен, если нужно прочитать именованные данные в выбранном столбце. Например, данные такого вида:
x,y,z
0,1,1
1,0,2
2,1,3
и т.д.

Параметры шаблона
TТипа выходного массива функции чтения файла
HBuffSizeколичество читаемых позиций файла

Читает данные из файла и записывает их в массив.

Аргументы
filenameИмя файла для чтения.
column_nameИмя столбца, из которого нужно получить данные.
resultМассив, в который будут записаны считанные данные.
nШаг между читаемыми строками.
delimiterРазделитель столбцов в файле.
Возвращает
int16_t Возвращает -1, если столбец не найден в заголовке файла.
Возвращает количество недочитанных строк, если файл закончился до достижения HBuffSize.
Возвращает 0 в случае успешного чтения всех строк.
