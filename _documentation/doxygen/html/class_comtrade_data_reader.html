Функция для считывания данных по каналам

  1. Заполням рабочий контейнер нулевыми векторами, чтобы иметь возможность обращаться к ним;
  2. Получаем строку;
  3. Получаем значения по строке и добавляем их в выделенные вектора.
    Каналы заполняются равномерно: новая строка - новое значение для каждого канала. Время парсинга значительно сокращается.
Параметры шаблона
TПараметр выборки данных
Аргументы
datFileСсылка на поток для чтения с файла данных
dataСсылка на контейннер для хранения информации о каналах (аналоговых или дискретных)
numChannelsЧисло каналов (аналоговых или дискретных)
delimiterРазделитель данных
