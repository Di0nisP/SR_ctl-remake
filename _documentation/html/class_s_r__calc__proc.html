◆ get_ready()

int SR_calc_proc::get_ready ( )

Метод для проверки готовности алгоритма

Алгоритм считается готовым, если все указатели на значения (входы и выходы), которые объявлены в алгоритме, не являются нулевыми, то есть все входы алгоритма подключены.

Возвращает
int Возвращает -1, если алгоритм не готов; возвращает 0, если алгоритм готов
