Подробное описание

Класс для обработки списков переменных

Переменные делятся на:

  1. выходы - значения, которые формируются локальными алгортмами;
  2. локальные входы - переменные, которые локальные алгоритмы получают от других алгоритмов;
  3. удалённые входы - переменные, которые удалённые алгоритмы получаютс от локальных алгоритмов.
    Класс управляет регистрацией локальных входов-выходов, выделением памяти под выходные переменные и связыванием входов-выходов.

Метод выделения памяти для связей входы-выходы всех алгоритмов

Cвязь выход-вход представляет собой область памяти на куче (выделяется с помощью операции new). Если несколько алгоритмов имеют указатели на эту общую область памяти, считается, что такие алгоритмы связаны, причём один из указателей этих алгоритмов является выходом. Память выделяется под выходы, так как выходы формируют значение.
