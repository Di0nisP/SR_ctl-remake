\doxysubsection{Подробное описание}
Класс для обработки списков переменных 

Переменные делятся на\+: ~\newline

\begin{DoxyEnumerate}
\item выходы -\/ значения, которые формируются локальными алгортмами; ~\newline

\item локальные входы -\/ переменные, которые локальные алгоритмы получают от других алгоритмов; ~\newline

\item удалённые входы -\/ переменные, которые удалённые алгоритмы получаютс от локальных алгоритмов. ~\newline
 Класс управляет регистрацией локальных входов-\/выходов, выделением памяти под выходные переменные и связыванием входов-\/выходов. 
\end{DoxyEnumerate}

Метод выделения памяти для связей входы-\/выходы всех алгоритмов 

Cвязь выход-\/вход представляет собой область памяти на куче (выделяется с помощью операции {\ttfamily new}).
